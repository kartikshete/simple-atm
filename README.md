# 💳 Simple ATM
A meticulously designed vanilla web application that successfully visualizes a real-world secure ATM network transaction experience.

## 📝 Overview
The Simple ATM project translates complex banking logic into an easily navigable interactive simulation. Complete with secure PIN validations, configurable dashboard states, real-time balance aggregations, and session history management, the application guarantees an authentic "cash withdrawal and deposit" experience using purely local state systems without external APIs.

## ✨ Key Features
- **Bank State Persistence**: Tracks balance, transaction ledgers, and profile configurations using high-performance Local Storage mechanisms.
- **PIN Security System**: Implements a numerical locking sequence that authenticates user interactions before menu access.
- **Dynamic Cash Flow**: Realistic logic preventing overdrafts while appending all approvals cleanly into the transaction UI.
- **Micro-animations**: Smooth overlays, modal pop-ups, and success ticks utilizing advanced CSS styling and SVGs.

## 🛠 Tech Stack
- Frontend: `HTML5`
- Styling: Custom `Vanilla CSS3`
- Operations Logic: `ES6 JavaScript`

## 🚀 Getting Started

This application does not rely on a build tool, thus starting it is incredibly straightforward.

```bash
# Clone the repository
cd simple-atm

# Open it directly in a modern browser
open index.html
```


## 🌐 Deployment

### Vercel / Netlify
1. This is a standalone static site.
2. Drag and drop the project folder into Netlify Drop or connect via GitHub.
3. No build command is required. Just ensure `index.html` is in the root.
